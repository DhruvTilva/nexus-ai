    Quick Guide

    How to use NexusAI from anywhere.

    ① The only thing you need

    Every request needs 2 things in the header:

    x-api-key : mykey1
    Content-Type : application/json

    Your API keys are set in .env → ALLOWED_API_KEYS

    ② Test in Postman

    1 Method: POST  |  URL: http://localhost:3000/ask
    2 Go to Headers tab → add: x-api-key = mykey1
    3 Go to Body → select raw → select JSON
    4 Paste this body and hit Send
    {
      "prompt": "What is AI?",
      "provider": "auto"
    }

    ③ Test in PowerShell

    PowerShell doesn't support Unix curl — use this instead:

    Invoke-RestMethod -Uri "http://localhost:3000/ask" `
      -Method POST `
      -Headers @{"Content-Type"="application/json"; "x-api-key"="mykey1"} `
      -Body '{"prompt": "What is AI?"}'

    ④ Test with curl (Git Bash / Linux / Mac)

    curl -X POST http://localhost:3000/ask \
      -H "Content-Type: application/json" \
      -H "x-api-key: mykey1" \
      -d '{"prompt": "What is AI?"}'

    ⑤ Use in any Website / JS app

    const res = await fetch("http://localhost:3000/ask", {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "x-api-key": "mykey1"
      },
      body: JSON.stringify({ prompt: "What is AI?" })
    });
    const data = await res.json();
    console.log(data.response); // AI answer here

    ⑥ Use in Python

    import requests

    res = requests.post("http://localhost:3000/ask",
      headers={"x-api-key": "mykey1"},
      json={"prompt": "What is AI?"}
    )
    print(res.json()["response"])

    ⑦ Understanding the Response

    {
      "success": true,
      "response": "AI is...",  ← your answer
      "provider": "gemini",  ← which AI answered
      "cached": false,  ← true = instant from cache
      "latency_ms": 1200  ← response time
    }

    ⑧ Optional Parameters — How to Change Them

    provider — which AI to use. Default: auto

    Pick one: gemini groq huggingface cohere cloudflare openrouter ollama

    {
      "prompt": "What is AI?",
      "provider": "groq"  ← force use Groq only
    }
    max_tokens — how long the answer can be. Default: 1000

    Lower = shorter answer, faster. Higher = longer answer.

    {
      "prompt": "Explain quantum computing",
      "max_tokens": 500  ← short answer
    }

    // or for a very detailed answer:
      "max_tokens": 3000
    temperature — creativity level. Default: 0.7
    0.0 – 0.3
    Factual, precise
    Good for: code, math
    0.5 – 0.7
    Balanced
    Good for: general Q&A
    0.8 – 1.0
    Creative, varied
    Good for: stories, ideas
    {
      "prompt": "Write a poem about space",
      "temperature": 0.9  ← more creative
    }
    cache — use cached response. Default: true

    Set to false if you always want a fresh answer (uses API quota).

    {
      "prompt": "What is today's news?",
      "cache": false  ← always fresh, skip cache
    }

    Smart AI API Router— v1.0.0

    One endpoint. Seven free AI providers. Zero cost. NexusAI routes your prompts intelligently across Gemini, Groq, HuggingFace, Cohere, Cloudflare AI, OpenRouter, and Ollama — with smart fallback, Redis caching, and request deduplication built in.

    • ✓ Redis response caching
    • ✓ Request deduplication
    • ✓ Health tracking per provider
    • ✓ API key authentication
    • ✓ Rate limiting (30 req/min)
